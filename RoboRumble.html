Robo Rumble: LAN Battle

Description:

RoboRumble: LAN Battle is an ambitious and immersive LAN-based multiplayer survival game designed for both Android and Windows platforms. The project aims to deliver an exhilarating gaming experience where players control robot characters in an enclosed arena, battling against each other to be the last one standing. The game incorporates advanced mechanics such as shooting, damaging, player controls, and player flying, which have already been successfully implemented. As a final year project, RoboRumble highlights my expertise in network systems and game development.

Features:

  • LAN Network Functionality: Enables players to engage in intense multiplayer battles within a localized network environment, ensuring low latency and stable connections.
  • Cross-Platform Compatibility: Seamless gameplay on both Android and Windows devices, optimizing performance across various hardware and screen resolutions.
  • Dynamic Combat Mechanics: Players can engage in thrilling battles with features such as shooting, attacking, and defensive maneuvers.
  • Power-Ups Integration: Diverse power-ups provide strategic advantages, enhancing gameplay dynamics without disrupting balance.
  • Spectator Mode: Allows eliminated players to observe ongoing matches, maintaining engagement and interaction within the game.

Current Progress:

The following components have been successfully developed:

  • Network System: Robust LAN network functionality for smooth multiplayer interactions.
  • Shooting Mechanics: Implemented precise and responsive shooting mechanisms.
  • Damage System: Comprehensive damage system to accurately reflect combat interactions.
  • Player Controls: Intuitive controls for player movement and actions.
  • Player Flying: Unique flying mechanics for enhanced mobility and strategic gameplay.

Technologies Used:

  • Unity Engine: Core game development platform.
  • C# Programming: Primary programming language for implementing game logic and mechanics.
  • Visual Studio: Integrated development environment for code development and debugging.
